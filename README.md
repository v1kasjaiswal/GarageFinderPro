Garage Finder Pro! is one of the most comprehensive and technically challenging apps📲 that I’ve built using Kotlin and XML layouts🧵. I developed this application as a part of my second-year project🎓 showcasing my skills in Android development. The app is focused on helpling users find garages in their vicinity, providing them with essential information📡 such as location, contact details, operating hours, detailed job card tracking, garage profile management, services offered, and customer reviews & ratings⭐. The app is designed to be user-friendly and efficient, ensuring a seamless experience for users looking for garages🚗. It reflects not just my technical abilities, but also my focus on building user-friendly applications that solve real-world problems👨‍🔧

PS - As Garage Finder Pro!, one of my earliest Android Projects🌱, it may not reflect all best practices⚙️ but it taught me a lot about real-world android app development✨ where every feature added and bug fixed is a learning experience😄

### **// Features⚡**

- **Authentication**: Role-based sign-up and sign-in using Firebase Authentication with Email/Password or Google Sign-In.
- **Database**: Storing user data in Firebase Firestore and Storage for real-time updates and secure data retrieval.
- **User Profile**: Users can manage their profiles, including adding vehicles and contact information.
- **Garage Owner Profile**: Garage owners can manage their profiles, including services offered, specializations, and contact details.
- **Garage Search**: Users can quickly find nearby garages with filters like service cost and towing service.
- **View Garage Details**: users can view garage details, including services offered, location, hours,ratings, and reviews.
- **Service Request**: Users can request services from garages, including details like vehicle type & service type.
- **Job Card Tracking**: Users can track their service requests with real-time updates on status and payments.
- **Garage Dashboard**: Garage owners can view all job requests including reviews, payments, analytics and service insights.
- **Real-Time Notifications**: Get real-time notification alerts for status updates, payments, and reviews.
- **Ratings & Reviews**: Share feedback and help others choose the right garage based on real user experiences.
- **Contact Us**: Users can contact the app support team for any queries or issues.

### **// XML Layout UI Components 🪨**

- **ConstraintLayout**: Used for building responsive, flexible, and flat hierarchy layouts.
- **RecyclerView**: Efficiently displays dynamic lists data using ViewHolder pattern.
- **CardView**: Displays information in a card format, providing a clean and modern look.
- **TextView**: Displays text in various styles, formats and with marquee effect.
- **TextInputLayout**: Provides a floating label for text input fields, enhancing user experience.
- **ImageView**: Displays images in various formats, sizes and shapes.
- **Buttons**: Displays clickable buttons for user actions, with various styles and effects.
- **BottomNavigationView**: Provides a bottom navigation bar for easy access to different app sections.
- **DialogBuilder**: Displays custom dialogs for user interactions, confirmations, and alerts.
- **BottomSheetDialog**: Displays a bottom sheet for additional options or information without leaving the current screen.

### **// Libraries and Dependencies 🤖**

- **Material Components**: Provides Material Design components for building beautiful and responsive UIs.
- **Coroutines**: Kotlin library for asynchronous programming, used for handling background tasks and network requests.
- **OkHttp**: HTTP client for Android, used for making network requests and handling responses.
- **Image Picker**: Library for selecting images from the device's gallery or camera.
- **Picasso**: Image loading and caching library for Android, used for displaying images efficiently.
- **Firebase Auth Library**: Official Firebase library for authentication services like sign-in, sign-up, email-verification and password reset.   
- **Firebase Firestore**: Official Firebase library for real-time database services, including data storage and retrieval.
- **Firebase Storage**: Official Firebase library for storing and retrieving files, including images and documents.
- **Firebase Cloud Messaging**: Official Firebase library for sending and receiving real-time notifications.
- **Notify**: Library for creating and managing notifications in Android apps.
- **MPAndroidChart**: Library for creating beautiful and interactive charts and graphs in Android apps.
- **Lottie**: Library for rendering animations in Android apps, used for creating beautiful and engaging UIs.
- **Balloon**: Library for creating beautiful and customizable balloons and tooltips in Android apps.

### **// Installation 📲**

- Clone the repository using the following command:
  ```bash
  // Clone the repository
  git clone https://github.com/v1kasjaiswal/GarageFinderPro.git

  // Change the directory
  cd GarageFinderPro

  // Make Sure to Add the google-services.json file in the app directory
  ```  
- Open the project in Android Studio and build the project.
- Run the project on an emulator or physical device.
